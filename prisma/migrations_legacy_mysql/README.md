# Legacy MySQL Migration Archive

Folder ini menyimpan migration chain lama yang dibuat saat provider Prisma masih MySQL.

- Status: arsip (tidak dipakai oleh `prisma migrate deploy` saat ini)
- Provider aktif saat ini: PostgreSQL
- Active migration chain: `prisma/migrations/`

Jangan menghapus folder ini tanpa backup, karena masih berguna untuk audit histori perubahan schema lama.
